PROGRAM EDITING:
	ADD TRACK
	MOVE TRACK LEFT
	MOVE TRACK RIGHT
	DUPLICATE TRACK
	REMOVE TRACK

GLOBAL EDITING:
	ADD BLOCK
	MOVE BLOCK
	COPY BLOCK
	COPY LINK BLOCK
	DELETE BLOCK

CURSOR EDITING:
	INSERT NOTE
	MODIFY NOTE
	MOVE NOTE
	DELETE NOTE
	CLEAR FIELD
	ETC (Misc commands that affect a block)

AUTOMATION EDITING
	ADD POINT
	MOVE POINT
	MODIFY POINT
	REMOVE POINT

SELECTION EDITING:
	COPY (goes to clipboard)
	PASTE OVERWRITE
	PASTE INSERT
	PASTE MIX
	PASTE MIX/REPLACING
	SELECTION OPERATIONS (like normal editing but on multiple tracks)

TRACK EDITING
	ADD AUTOMATION/HIDE AND REMOVE AUTOMATION
	MOVE AUTOMATION LEFT AND RIGHT
	MODIFY PARAMETER (PROPERTY)
	ADD/REMOVE SYNTH (music tracks only)
	ADD/REMOVE/MOVE EFFECT
	CONNECT PLUGS
	DISCONNECT PLUGS
